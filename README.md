# text2color-green

A simple Node.js module to change text color to green.

## Installation

\`\`\`bash
npm install text2color-green
\`\`\`

## Usage

\`\`\`javascript
const textToGreen = require('text2color-green');

console.log(textToGreen('This text will be green!'));
\`\`\`
